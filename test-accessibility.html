Test Content

This is a test page for the Starlet25 accessibility feature. When you press Alt+N, this content should be read aloud using text-to-speech.

The accessibility feature should:

  • Respond to the Alt+N keyboard shortcut
  • Show a visual indicator when reading
  • Read the main content of the page
  • Provide console logging for debugging

If the Alt+N shortcut is not working, check the browser console for error messages and ensure the extension is properly loaded.
