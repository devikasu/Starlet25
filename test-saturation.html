üìù Sample Content

This is sample content with red text, green text, and blue text to demonstrate how text colors are affected by the saturation filter.

The saturation filter applies to the entire webpage using CSS filter: saturate(). This can be useful for:

  • Accessibility - Reducing saturation for users with visual sensitivities
  • Reading comfort - Muted colors can reduce eye strain
  • Visual appeal - Increased saturation for more vibrant content
  • Color correction - Adjusting for different display settings

üîß Technical Details

How it works:
  • The extension injects a CSS filter into the webpage
  • The filter is applied to the <html> element
  • Values range from 0% (grayscale) to 200% (very saturated)
  • 100% is the default/normal saturation
  • The filter affects all colors on the page including images
